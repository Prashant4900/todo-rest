# Django Todo App 
Simple Django app to manage todo list.

<hr>

## Requirements
- [Django](https://www.djangoproject.com/)
- [Python](https://www.python.org/)

## Installation
    
    git clone https://github.com/Prashant4900/todo-rest
    cd todo-rest
    python3 manage.py migrate
    python3 manage.py runserver
